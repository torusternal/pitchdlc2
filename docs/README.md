# Zenith - Premium Software Website

Копия сайта https://zenithdlc.org/ с полным функционалом и дизайном.

## Структура проекта

- `index.html` - Главная страница с hero-секцией, статистикой, отзывами и видео
- `login.html` - Страница авторизации
- `register.html` - Страница регистрации
- `profile.html` - Личный кабинет пользователя
- `products.html` - Страница с тарифами и ценами
- `faq.html` - Часто задаваемые вопросы
- `support.html` - Форма поддержки
- `style.css` - Основные стили (glassmorphism, анимации, адаптивность)

## Особенности дизайна

- Тёмная тема с черным фоном (#000)
- Glassmorphism эффекты (blur, прозрачность)
- Анимированные фоновые волны
- Floating анимации для изображений
- Hover-эффекты на карточках
- Адаптивная верстка для мобильных устройств
- Мобильное меню (бургер)

## Технологии

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- JavaScript (Vanilla)
- SVG графика

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
cd docs
python -m http.server 8080
```
